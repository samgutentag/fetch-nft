# Feedback and Support

## Direct Support:

Longer term goal to create funnel to fewer support channels, consolidating developers to a place with more information and other builders

## Community Support:

Create a “Verified Developer” role, empowering and recognizing the efforts of existing developers to grow the ecosystem and help others

## Regular Feedback Sessions:

Host regular sessions providing a platform for developers to interact with Audius DevRel, engineering and product teams directly

## Bug Bounty Program:

Encourage developers to report security vulnerabilities, bugs, and enhancements with $AUDIO rewards
