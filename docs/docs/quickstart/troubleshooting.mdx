# Troubleshooting

> Having Issues? Here are some commonly reported issues and resolutions.

## Examples not returning Ethereum NFTs

> By default, Fetch-NFT uses the public Opensea API and the Solana mainnet RPC endpoint. To configure API keys and endpoints, see [Usage With Configs](./config-usage).

## React Example App Certificate Errors

Changes to react have introduced some certificat issues. While not suggested for production apps, making the below change to your `package.json` file will get you up and running.

replace this:

```json
"start": "react-scripts start",
```

with this:

```json
"start": "react-scripts --openssl-legacy-provider start",
```
